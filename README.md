# OmniPanel - 全能面板

## 一体化服务器&游戏管理桌面应用

基于 Electron + Vue 3 + TypeScript + Tailwind CSS + Pinia 构建的跨平台桌面应用程序。

## 开发

```bash
# 安装依赖
npm install

# 启动开发服务器 (前端)
npm run dev

# 启动后端 API 服务
npm run server

# 同时启动前端和后端
npm run dev:all

# 构建生产版本
npm run build
```

## 技术栈

- **前端**: Electron 30 + Vue 3 + TypeScript + Element Plus + Tailwind CSS + Pinia + ECharts
- **后端**: Node.js + Express + SQLite + WebSocket
- **终端**: xterm.js
- **代码编辑器**: Monaco Editor
